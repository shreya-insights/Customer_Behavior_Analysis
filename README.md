# Data Analytics Project

## Overview
This project focuses on performing end-to-end data analytics using Python, SQL, and Power BI. The workflow includes data loading, data cleaning, exploratory data analysis (EDA), SQL-based analysis, dashboard creation, and business reporting.

The main objective of this project is to transform raw data into meaningful insights that help in better business decision-making.

---

## Dataset
The dataset contains customer and transaction-related information such as:

- Customer details
- Purchase behavior
- Product categories
- Payment methods
- Subscription status
- Ratings and reviews
- Revenue-related information

The dataset was analyzed to identify trends, customer behavior, and business insights.

## Project Workflow

### 1. Data Loading
- Imported dataset using Python
- Checked dataset structure and column information
- Handled missing and duplicate values

### 2. Exploratory Data Analysis (EDA)
Performed:
- Summary statistics
- Data distribution analysis
- Correlation analysis
- Category-wise analysis
- Customer behavior analysis

Created visualizations such as:
- Bar charts
- Pie charts
- Histograms
- Heatmaps
- Line charts

### 3. Data Cleaning
- Removed duplicates
- Handled null values
- Corrected data types
- Standardized column values
- Created derived columns where required

### 4. SQL Analysis
Performed SQL analysis using PostgreSQL/MySQL/SQL Server:
- Aggregate functions
- Joins
- Group By & Having
- Window functions
- CTEs
- Revenue analysis
- Customer segmentation queries
- Trend analysis

### 5. Power BI Dashboard
Built an interactive dashboard including:
- KPI cards
- Donut charts
- Bar charts
- Revenue trends
- Customer segmentation
- Subscription analysis
- Filters and slicers

### 6. Reporting & Presentation
- Created a business report summarizing findings
- Designed presentation slides using Gamma AI
- Presented insights and recommendations visually

## Key Insights

Some major insights identified during the analysis:

- Certain product categories generated higher revenue
- Repeat customers showed higher subscription rates
- Purchase behavior varied across age groups
- Customer ratings influenced purchasing frequency
- Seasonal trends impacted sales performance
